### 📌 ES6 이전의 리스트 순회
```javascript
// 일반 배열
var list = [1, 2, 3, 4, 5];
for (var i = 0; i < list.length; i++) {
  console.log(list[i]); // 1 2 3 4 5
}

// 유사 배열
var str = 'abc';
for (var i = 0; i < str.length; i++) {
  console.log(str[i]); // a b c
}
```

### 📌 ES6 이후의 리스트 순회
```javascript
// 일반 배열
const list = [1, 2, 3, 4, 5];
for (const i of list) {
  console.log(i); // 1 2 3 4 5
}

// 유사 배열
const str = 'abc';
for (const i of str) {
  console.log(i); // a b c
}
```
